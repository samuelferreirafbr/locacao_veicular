Informacoes basicas: Esse sistema tem como objetivo manipular dado de uma locadora de carros.

Modo de uso:

[x] Instalacao:

- Apos extrair os arquivos, mova a pasta 'locacaoVeicular' para a pasta
htdocs (dentro das pastas do XAMPP no disco local)

[x] Para configurar o Banco de Dados:

- Execute o script do banco de dados;
- Da star no xampp no Apache, MySQL, FileZilla e Mercury;
- Abra o Admin no MySQL;
- No navegador abra a aba de "banco de dados" e clica no db_locadoraVeicular;

[x] Para manipular dados:

- Inserir User:
	http://localhost/locacaoVeicular/inserirUser.php
- Inserir Veiculos:
	http://localhost/locacaoVeicular/inserirVeiculo.php
- Inserir Emprestimo:
	http://localhost/locacaoVeicular/inserirEmprestimo.php
- Consultar e alterar dados:
	http://localhost/locacaoVeicular/alteracao.php
